StreakXP Trading Trainer - Account and Data Deletion

This page explains how users can request account deletion and how they can delete only part of their data.

Last updated: 2026-04-13

Full Account Deletion

Delete account and all related data

Users can request complete account deletion directly from the app support channel.

  • Send a request email to: webmaniabusiness@gmail.com
  • Include the email used for login (Google account)
  • Use subject: Account Deletion Request

On successful verification, the account and related cloud data are permanently removed.

Partial Data Deletion

Delete only profile/challenge data (without deleting account)

The app supports in-app partial deletion.

  • Open Profiles
  • Select the profile you want to remove
  • Use the Delete Profile action

This removes profile-related progress data (missions, challenge history, profile state) while keeping the user account active.

This is a supported partial deletion flow.

Retention

Data retention policy

  • Full account deletion requests are processed within 30 days.
  • After deletion, recovery is not guaranteed.
  • Minimal technical logs may be retained for legal/security obligations.
Important

What this means for Google Play form

  • Can users request account deletion? Yes
  • Do users have partial deletion without deleting account? Yes (profile deletion in-app)
