StreakXP Trading Trainer - Data Deletion Request

This page describes how users can request deletion of their app data for StreakXP Trading Trainer.

How users can request deletion

  1. Open the app.
  2. Go to Profile.
  3. Choose one of the following options:
  • Delete a single profile: use the trash icon next to the selected profile in the profile modal.
  • Full local reset: tap Reset totale dati locali to remove all local app data.

Data that is deleted

  • Local user profiles
  • XP, levels, streaks
  • Challenges and missions progress
  • Trade history and local app state/preferences

Data that is not deleted by this request

  • Google Play purchase and subscription records (managed by Google Play)
  • Billing entitlement history linked to the user's Google account

Retention period

Data removed with profile deletion or full local reset is deleted immediately on device.

Need support

If you cannot access the app and still need help, contact the developer using the support channel listed on the Google Play store page for StreakXP Trading Trainer.
